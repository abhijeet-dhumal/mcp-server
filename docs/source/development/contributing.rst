Contributing
============

Full guidelines: `CONTRIBUTING.md <https://github.com/kubeflow/mcp-server/blob/main/CONTRIBUTING.md>`_

Quick Start
-----------

.. code-block:: bash

   pip install -e ".[dev]"
   make test
   make lint

Code Style
----------

- PEP 8 + Google-style docstrings
- Format with ``ruff format``
- Type hints for public APIs
